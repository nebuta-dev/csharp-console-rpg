C# Console RPG

A text-based RPG built in C# to practice object-oriented programming and core software development concepts.

Features
- Player and Creature class hierarchy
- Abstract base classes and method overriding
- Inventory and stats management
- Turn-based combat loop
- Menu-driven user interaction
- Console UI elements (stat bars, ASCII art)

Concepts Practiced
- Object-Oriented Programming (OOP)
- Encapsulation, inheritance, polymorphism
- Control flow and state management
- Input validation and error handling

How to Run
- Clone the repository
- Open in Visual Studio or VS Code
- Run the project from the console
